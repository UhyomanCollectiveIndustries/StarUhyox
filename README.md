# UhyomanCollectiveIndustries
## StarUhyox
OpenGLを使った3D描画での3Dシューティングゲームです。

### 環境

###### ライブラリ:
Path:LearnOpenGL\libs  
**Glad(OpenGL関数ローダー):**  
https://glad.dav1d.de/#language=c\&specification=gl\&api=gl%3D3.3\&api=gles1%3Dnone\&api=gles2%3Dnone\&api=glsc2%3Dnone\&profile=core\&loader=on  
からダウンロード  
libs\glad\~   
  
**Glfw(ウィンドウ作成・入力処理):**
https://www.glfw.org/download.html  
からダウンロード  
libs\glfw\~  
  
**GLM(数学ライブラリ)**
https://github.com/g-truc/glm
からダウンロード  
libs\glm\~

CMake
